# Leap & Flit

Fast, low-latency jump and enhanced `f`/`t` motions.

## Plugins
- `ggandor/leap.nvim` – cross-window jump with minimal keystrokes.
- `ggandor/flit.nvim` – upgrades `f`/`t` motions with repeat and label support.

## Default Keymaps
- `s` – jump to any target in the current window.
- `S` – jump across windows.
- `f`, `F`, `t`, `T` – enhanced single-character motions.

## Tips
- Use `;` and `,` to repeat the last `f`/`t` motion (Flit enhances this flow).
- For alternative mapping styles, see `:help leap-mappings`.
